# Order Summary Component

This is a solution to the [Order summary component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/order-summary-component-QmSp6dfYJ).

## 📋 Project Description
A clean order summary card component featuring a music subscription plan layout. This challenge focuses on perfecting card layouts, typography, hover states for interactive elements, and handling responsive background illustrations.

## 🚀 Demo
[Live Preview](https://fokasdev.github.io/Order-summary-component/)

## 🛠️ Built With
* **Semantic HTML5** – For structured and accessible markup.
* **CSS3 Custom Properties** – Using variables for consistent layout coloring.
* **Flexbox** – For centering the component and aligning card contents.
* **Mobile-first workflow** – Ensuring responsiveness across all viewports.
